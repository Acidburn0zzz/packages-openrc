###Build.sh###

    Usage: build.sh [options]
        -h                   This help
        -a <arch>            Set arch
        -b <branch>          Set branch
        -c                   Recreate the chroot before building
        -p <profile>         Set profile or pkg
        -r <dir>             Create chroots in this directory
        -s                   Sign packages
